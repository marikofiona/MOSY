How-To:

Ihr könnte die angehängten Dateien auf zwei Arten verwenden:

1. Vergleicht euren Code mit dem Lösungscode (hauptsächlich die JavaScript Dateien)

2. Führt den Lösungscode aus: öffnet nach dem Extrahieren das Command Promt/Terminal, navigiert in den Lösungsordner (mit cd command) und führt die Befehle "npm install" und anschließend "npm run start" aus.

Das Projekt wurde getestet mit:
Node.js 14.6.1
Expo CLI 48
React Native 0.71
React 18.0
